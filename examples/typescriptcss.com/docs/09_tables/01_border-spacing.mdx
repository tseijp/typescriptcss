## border-spacing

| Utility                       | Styles                                                                          |
| ----------------------------- | ------------------------------------------------------------------------------- |
| `border.spacing[*<value>*]`   | `border-spacing: calc(var(--spacing) * *<number>*);`                            |
| `border.spacing.x[*<value>*]` | `border-spacing: calc(var(--spacing) * *<number>*) var(--tw-border-spacing-y);` |
| `border.spacing.y[*<value>*]` | `border-spacing: var(--tw-border-spacing-x) calc(var(--spacing) * *<number>*);` |
