## blur

Utilities for applying backdrop blur filters to an element.

| Utility                    | Styles                                             |
| -------------------------- | -------------------------------------------------- |
| `backdrop.blur[*<value>*]` | `backdrop-filter: blur(var(--blur-xs)); /* 4px */` |
| `backdrop.blur.none`       | `backdrop-filter: ;`                               |
