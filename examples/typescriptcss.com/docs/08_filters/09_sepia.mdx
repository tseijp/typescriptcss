## sepia

Utilities for applying sepia filters to an element.

| Utility            | Styles                        |
| ------------------ | ----------------------------- |
| `sepia`            | `filter: sepia(100%);`        |
| `sepia[*<value>*]` | `filter: sepia(*<number>*%);` |
