## backdrop-filter

Utilities for applying backdrop filters to an element.

| Utility                      | Styles                                       |
| ---------------------------- | -------------------------------------------- |
| `backdrop.filter.none`       | `backdrop-filter: none;`                     |
| `backdrop.filter[*<value>*]` | `backdrop-filter: var(*<custom-property>*);` |
