## grayscale

Utilities for applying backdrop grayscale filters to an element.

| Utility                         | Styles                                     |
| ------------------------------- | ------------------------------------------ |
| `backdrop.grayscale`            | `backdrop-filter: grayscale(100%);`        |
| `backdrop.grayscale[*<value>*]` | `backdrop-filter: grayscale(*<number>*%);` |
