## drop-shadow

Utilities for applying drop-shadow filters to an element.

| Utility                  | Styles                                                                          |
| ------------------------ | ------------------------------------------------------------------------------- |
| `drop.shadow[*<value>*]` | `filter: drop-shadow(var(--drop-shadow-xs)); /* 0 1px 1px rgb(0 0 0 / 0.05) */` |
