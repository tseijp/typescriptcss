## grayscale

Utilities for applying grayscale filters to an element.

| Utility                | Styles                            |
| ---------------------- | --------------------------------- |
| `grayscale`            | `filter: grayscale(100%);`        |
| `grayscale[*<value>*]` | `filter: grayscale(*<number>*%);` |
