## blur

Utilities for applying blur filters to an element.

| Utility           | Styles                                    |
| ----------------- | ----------------------------------------- |
| `blur[*<value>*]` | `filter: blur(var(--blur-xs)); /* 4px */` |
| `blur.none`       | `filter: ;`                               |
