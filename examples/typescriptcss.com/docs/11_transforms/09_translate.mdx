## translate

Utilities for translating elements.

| Utility                   | Styles                                                                                       |
| ------------------------- | -------------------------------------------------------------------------------------------- |
| `translate[*<value>*]`    | `translate: calc(var(--spacing) * *<number>*) calc(var(--spacing) * *<number>*);`            |
| `-translate[*<value>*]`   | `translate: calc(var(--spacing) * -*<number>*) calc(var(--spacing) * -*<number>*);`          |
| `translate.full`          | `translate: 100% 100%;`                                                                      |
| `-translate.full`         | `translate: -100% -100%;`                                                                    |
| `translate.px`            | `translate: 1px 1px;`                                                                        |
| `-translate.px`           | `translate: -1px -1px;`                                                                      |
| `translate.none`          | `translate: none;`                                                                           |
