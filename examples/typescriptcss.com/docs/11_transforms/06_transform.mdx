## transform

Utilities for transforming elements.

| Utility                | Styles                                                                                                                 |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `transform[*<value>*]` | `transform: var(*<custom-property>*);`                                                                                 |
| `transform.none`       | `transform: none;`                                                                                                     |
