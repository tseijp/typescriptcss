## transform-style

Utilities for controlling if an elements children are placed in 3D space.

| Utility           | Styles                          |
| ----------------- | ------------------------------- |
| `transform['3d']` | `transform-style: preserve-3d;` |
| `transform.flat`  | `transform-style: flat;`        |
