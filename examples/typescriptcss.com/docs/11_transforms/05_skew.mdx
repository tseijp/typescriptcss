## skew

Utilities for skewing elements with transform.

| Utility              | Styles                                                    |
| -------------------- | --------------------------------------------------------- |
| `skew[*<value>*]`    | `transform: skewX(*<number>*deg) skewY(*<number>*deg);`   |
| `-skew[*<value>*]`   | `transform: skewX(-*<number>*deg) skewY(-*<number>*deg);` |
| `skew.x[*<value>*]`  | `transform: skewX(*<number>*deg));`                       |
| `-skew.x[*<value>*]` | `transform: skewX(-*<number>*deg));`                      |
| `skew.y[*<value>*]`  | `transform: skewY(*<number>*deg);`                        |
| `-skew.y[*<value>*]` | `transform: skewY(-*<number>*deg);`                       |
