## backface-visibility

Utilities for controlling if an element's backface is visible.

| Utility            | Styles                          |
| ------------------ | ------------------------------- |
| `backface.hidden`  | `backface-visibility: hidden;`  |
| `backface.visible` | `backface-visibility: visible;` |
