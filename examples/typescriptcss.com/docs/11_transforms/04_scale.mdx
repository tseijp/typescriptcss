## scale

Utilities for scaling elements.

| Utility               | Styles                                                               |
| --------------------- | -------------------------------------------------------------------- |
| `scale.none`          | `scale: none;`                                                       |
| `scale[*<value>*]`    | `scale: *<number>*% *<number>*%;`                                    |
| `-scale[*<value>*]`   | `scale: calc(*<number>*% * -1) calc(*<number>*% * -1);`              |
| `scale.x[*<value>*]`  | `scale: *<number>*% var(--tw-scale-y);`                              |
| `-scale.x[*<value>*]` | `scale: calc(*<number>*% * -1) var(--tw-scale-y);`                   |
| `scale.y[*<value>*]`  | `scale: var(--tw-scale-x) *<number>*%;`                              |
| `-scale.y[*<value>*]` | `scale: var(--tw-scale-x) calc(*<number>*% * -1);`                   |
| `scale.z[*<value>*]`  | `scale: var(--tw-scale-x) var(--tw-scale-y) *<number>*%;`            |
| `-scale.z[*<value>*]` | `scale: var(--tw-scale-x) var(--tw-scale-y) calc(*<number>*% * -1);` |
| `scale['3d']`         | `scale: var(--tw-scale-x) var(--tw-scale-y) var(--tw-scale-z);`      |
