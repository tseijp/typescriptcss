## flex-basis

Utilities for controlling the initial size of flex items.

| Utility            | Styles                                           |
| ------------------ | ------------------------------------------------ |
| `basis[*<value>*]` | `flex-basis: calc(var(--spacing) * *<number>*);` |
| `basis.full`       | `flex-basis: 100%;`                              |
| `basis.auto`       | `flex-basis: auto;`                              |
