## gap

Utilities for controlling gutters between grid and flexbox items.

| Utility            | Styles                                          |
| ------------------ | ----------------------------------------------- |
| `gap[*<value>*]`   | `gap: calc(var(--spacing) * *<value>*);`        |
| `gap.x[*<value>*]` | `column-gap: calc(var(--spacing) * *<value>*);` |
| `gap.y[*<value>*]` | `row-gap: calc(var(--spacing) * *<value>*);`    |
