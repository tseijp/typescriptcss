## align-items

Utilities for controlling how flex and grid items are positioned along a container's cross axis.

| Utility                  | Styles                        |
| ------------------------ | ----------------------------- |
| `items['flex-start']`    | `align-items: flex-start;`    |
| `items['flex-end']`      | `align-items: flex-end;`      |
| `items['safe flex-end']` | `align-items: safe flex-end;` |
| `items.center`           | `align-items: center;`        |
| `items['safe center']`   | `align-items: safe center;`   |
| `items.baseline`         | `align-items: baseline;`      |
| `items['last baseline']` | `align-items: last baseline;` |
| `items.stretch`          | `align-items: stretch;`       |
