## grid-row

Utilities for controlling how elements are sized and placed across grid rows.

| Utility                 | Styles                                         |
| ----------------------- | ---------------------------------------------- |
| `row.span[*<value>*]`   | `grid-row: span *<number>* / span *<number>*;` |
| `row['1 / -1']`         | `grid-row: 1 / -1;`                            |
| `row.start[*<value>*]`  | `grid-row-start: *<number>*;`                  |
| `-row.start[*<value>*]` | `grid-row-start: calc(*<number>* * -1);`       |
| `row.start.auto`        | `grid-row-start: auto;`                        |
| `row.end[*<value>*]`    | `grid-row-end: *<number>*;`                    |
| `-row.end[*<value>*]`   | `grid-row-end: calc(*<number>* * -1);`         |
| `row.end.auto`          | `grid-row-end: auto;`                          |
| `row.auto`              | `grid-row: auto;`                              |
| `row[*<value>*]`        | `grid-row: *<number>*;`                        |
| `-row[*<value>*]`       | `grid-row: calc(*<number>* * -1);`             |
