## flex-grow

Utilities for controlling how flex items grow.

| Utility           | Styles                   |
| ----------------- | ------------------------ |
| `grow['1']`       | `flex-grow: 1;`          |
| `grow[*<value>*]` | `flex-grow: *<number>*;` |
