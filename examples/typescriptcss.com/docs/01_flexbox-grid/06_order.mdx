## order

Utilities for controlling the order of flex and grid items.

| Utility            | Styles               |
| ------------------ | -------------------- |
| `order[*<value>*]` | `order: *<number>*;` |
| `order['-9999']`   | `order: -9999;`      |
| `order['9999']`    | `order: 9999;`       |
| `order['0']`       | `order: 0;`          |
