## grid-template-rows

Utilities for specifying the rows in a grid layout.

| Utility                | Styles                                                    |
| ---------------------- | --------------------------------------------------------- |
| `grid.rows[*<value>*]` | `grid-template-rows: repeat(*<number>*, minmax(0, 1fr));` |
| `grid.rows.none`       | `grid-template-rows: none;`                               |
| `grid.rows.subgrid`    | `grid-template-rows: subgrid;`                            |
