## flex

Utilities for controlling how flex items both grow and shrink.

| Utility           | Styles              |
| ----------------- | ------------------- |
| `flex[*<value>*]` | `flex: *<number>*;` |
| `flex.auto`       | `flex: auto;`       |
| `flex['0 auto']`  | `flex: 0 auto;`     |
| `flex.none`       | `flex: none;`       |
