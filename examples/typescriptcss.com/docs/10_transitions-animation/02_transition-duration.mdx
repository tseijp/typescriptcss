## transition-duration

Utilities for controlling the duration of CSS transitions.

| Utility               | Styles                               |
| --------------------- | ------------------------------------ |
| `duration[*<value>*]` | `transition-duration: *<number>*ms;` |
| `duration.initial`    | `transition-duration: initial;`      |
