## visibility

Utilities for controlling the visibility of an element.

| Utility               | Styles                  |
| --------------------- | ----------------------- |
| `visibility.visible`  | `visibility: visible;`  |
| `visibility.hidden`   | `visibility: hidden;`   |
| `visibility.collapse` | `visibility: collapse;` |
