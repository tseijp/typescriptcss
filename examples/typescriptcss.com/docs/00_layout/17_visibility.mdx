## visibility

Utilities for controlling the visibility of an element.

| Utility     | Styles                  |
| ----------- | ----------------------- |
| `visible`   | `visibility: visible;`  |
| `invisible` | `visibility: hidden;`   |
| `collapse`  | `visibility: collapse;` |
