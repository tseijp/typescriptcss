## aspect-ratio

Utilities for controlling the aspect ratio of an element.

| Utility             | Styles                                            |
| ------------------- | ------------------------------------------------- |
| `aspect[*<value>*]` | `aspect-ratio: *<ratio>*;`                        |
| `aspect['1 / 1']`   | `aspect-ratio: 1 / 1;`                            |
| `aspect.video`      | `aspect-ratio: var(--aspect-video); /* 16 / 9 */` |
| `aspect.auto`       | `aspect-ratio: auto;`                             |
