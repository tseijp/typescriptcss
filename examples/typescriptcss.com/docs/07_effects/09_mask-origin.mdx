## mask-origin

Utilities for controlling how an element's mask image is positioned relative to borders, padding, and content.

| Utility                      | Styles                      |
| ---------------------------- | --------------------------- |
| `mask.origin['border-box']`  | `mask-origin: border-box;`  |
| `mask.origin['padding-box']` | `mask-origin: padding-box;` |
| `mask.origin['content-box']` | `mask-origin: content-box;` |
| `mask.origin['fill-box']`    | `mask-origin: fill-box;`    |
| `mask.origin['stroke-box']`  | `mask-origin: stroke-box;`  |
| `mask.origin['view-box']`    | `mask-origin: view-box;`    |
