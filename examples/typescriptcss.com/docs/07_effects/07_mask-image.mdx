## mask-image

Utilities for controlling an element's mask image.

| Utility                       | Styles                                                                                                                                                                                                                                                      |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `mask[*<value>*]`             | `mask-image: *<value>*;`                                                                                                                                                                                                                                    |
| `mask.none`                   | `mask-image: none;`                                                                                                                                                                                                                                         |
| `mask.radial[*<value>*]`      | `mask-image: radial-gradient(*<value>*);`                                                                                                                                                                                                                   |
