## mask-mode

Utilities for controlling an element's mask mode.

| Utility          | Styles                     |
| ---------------- | -------------------------- |
| `mask.alpha`     | `mask-mode: alpha;`        |
| `mask.luminance` | `mask-mode: luminance;`    |
| `mask.match`     | `mask-mode: match-source;` |
