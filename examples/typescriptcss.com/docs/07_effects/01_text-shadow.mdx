## text-shadow

Utilities for controlling the shadow of a text element.

| Utility                  | Styles                                                                     |
| ------------------------ | -------------------------------------------------------------------------- |
| `text.shadow[*<value>*]` | `text-shadow: var(--text-shadow-2xs); /* 0px 1px 0px rgb(0 0 0 / 0.15) */` |
| `text.shadow.none`       | `text-shadow: none;`                                                       |
