## box-shadow

Utilities for controlling the box shadow of an element.

| Utility               | Styles                                                         |
| --------------------- | -------------------------------------------------------------- |
| `shadow[*<value>*]`   | `box-shadow: var(--shadow-2xs); /* 0 1px rgb(0 0 0 / 0.05) */` |
| `shadow['0 0 #0000']` | `box-shadow: 0 0 #0000;`                                       |
