## mask-size

Utilities for controlling the size of an element's mask image.

| Utility                | Styles                                 |
| ---------------------- | -------------------------------------- |
| `mask.auto`            | `mask-size: auto;`                     |
| `mask.cover`           | `mask-size: cover;`                    |
| `mask.contain`         | `mask-size: contain;`                  |
| `mask.size[*<value>*]` | `mask-size: var(*<custom-property>*);` |
