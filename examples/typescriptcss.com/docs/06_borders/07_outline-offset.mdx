## outline-offset

Utilities for controlling the offset of an element's outline.

| Utility                      | Styles                                     |
| ---------------------------- | ------------------------------------------ |
| `outline.offset[*<value>*]`  | `outline-offset: *<number>*px;`            |
| `-outline.offset[*<value>*]` | `outline-offset: calc(*<number>*px * -1);` |
