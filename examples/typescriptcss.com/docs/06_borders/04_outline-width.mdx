## outline-width

Utilities for controlling the width of an element's outline.

| Utility              | Styles                         |
| -------------------- | ------------------------------ |
| `outline['1px']`     | `outline-width: 1px;`          |
| `outline[*<value>*]` | `outline-width: *<number>*px;` |
