## min-block-size

Utilities for setting the minimum block size of an element.

| Utility                    | Styles                                               |
| -------------------------- | ---------------------------------------------------- |
| `min.block[*<value>*]`     | `min-block-size: calc(var(--spacing) * *<number>*);` |
| `min.block['1px']`         | `min-block-size: 1px;`                               |
| `min.block['100%']`        | `min-block-size: 100%;`                              |
| `min.block['100vh']`       | `min-block-size: 100vh;`                             |
| `min.block['100dvh']`      | `min-block-size: 100dvh;`                            |
| `min.block['100dvw']`      | `min-block-size: 100dvw;`                            |
| `min.block['100lvh']`      | `min-block-size: 100lvh;`                            |
| `min.block['100lvw']`      | `min-block-size: 100lvw;`                            |
| `min.block['100svw']`      | `min-block-size: 100svw;`                            |
| `min.block['100svh']`      | `min-block-size: 100svh;`                            |
| `min.block.auto`           | `min-block-size: auto;`                              |
| `min.block['min-content']` | `min-block-size: min-content;`                       |
| `min.block['max-content']` | `min-block-size: max-content;`                       |
| `min.block['fit-content']` | `min-block-size: fit-content;`                       |
| `min.block['1lh']`         | `min-block-size: 1lh;`                               |
