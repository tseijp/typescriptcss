## min-block-size

Utilities for setting the minimum block size of an element.

| Utility                   | Styles                                               |
| ------------------------- | ---------------------------------------------------- |
| `min.block[*<value>*]`    | `min-block-size: calc(var(--spacing) * *<number>*);` |
| `min.size['1px']`         | `min-block-size: 1px;`                               |
| `min.size['100%']`        | `min-block-size: 100%;`                              |
| `min.size['100vh']`       | `min-block-size: 100vh;`                             |
| `min.size['100dvh']`      | `min-block-size: 100dvh;`                            |
| `min.size['100dvw']`      | `min-block-size: 100dvw;`                            |
| `min.size['100lvh']`      | `min-block-size: 100lvh;`                            |
| `min.size['100lvw']`      | `min-block-size: 100lvw;`                            |
| `min.size['100svw']`      | `min-block-size: 100svw;`                            |
| `min.size['100svh']`      | `min-block-size: 100svh;`                            |
| `min.size.auto`           | `min-block-size: auto;`                              |
| `min.size['min-content']` | `min-block-size: min-content;`                       |
| `min.size['max-content']` | `min-block-size: max-content;`                       |
| `min.size['fit-content']` | `min-block-size: fit-content;`                       |
| `min.size['1lh']`         | `min-block-size: 1lh;`                               |
