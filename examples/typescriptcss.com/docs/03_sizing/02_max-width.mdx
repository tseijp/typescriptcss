## max-width

Utilities for setting the maximum width of an element.

| Utility                | Styles                                                                                                                                                                                                                                          |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `max.w[*<value>*]`     | `max-width: calc(var(--spacing) * *<number>*);`                                                                                                                                                                                                 |
| `max.w.none`           | `max-width: none;`                                                                                                                                                                                                                              |
| `max.w['1px']`         | `max-width: 1px;`                                                                                                                                                                                                                               |
| `max.w['100%']`        | `max-width: 100%;`                                                                                                                                                                                                                              |
| `max.w['100dvw']`      | `max-width: 100dvw;`                                                                                                                                                                                                                            |
| `max.w['100dvh']`      | `max-width: 100dvh;`                                                                                                                                                                                                                            |
| `max.w['100lvw']`      | `max-width: 100lvw;`                                                                                                                                                                                                                            |
| `max.w['100lvh']`      | `max-width: 100lvh;`                                                                                                                                                                                                                            |
| `max.w['100svw']`      | `max-width: 100svw;`                                                                                                                                                                                                                            |
| `max.w['100svh']`      | `max-width: 100svh;`                                                                                                                                                                                                                            |
| `max.w['100vw']`       | `max-width: 100vw;`                                                                                                                                                                                                                             |
| `max.w['min-content']` | `max-width: min-content;`                                                                                                                                                                                                                       |
| `max.w['max-content']` | `max-width: max-content;`                                                                                                                                                                                                                       |
| `max.w['fit-content']` | `max-width: fit-content;`                                                                                                                                                                                                                       |
| `container`            | `width: 100%;@media (width >= 40rem) { max-width: 40rem; }@media (width >= 48rem) { max-width: 48rem; }@media (width >= 64rem) { max-width: 64rem; }@media (width >= 80rem) { max-width: 80rem; }@media (width >= 96rem) { max-width: 96rem; }` |
