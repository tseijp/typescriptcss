## height

Utilities for setting the height of an element.

| ----------------------- | ------------------------------------------------------------------------------------- |
| `h[*<value>*]`          | `height: calc(var(--spacing) * *<number>*);`                                          |

| `height['1px']`         | `height: 1px;`                                                                        |
| `height['100%']`        | `height: 100%;`                                                                       |
| `height['100vh']`       | `height: 100vh;`                                                                      |
| `height['100dvh']`      | `height: 100dvh;`                                                                     |
| `height['100dvw']`      | `height: 100dvw;`                                                                     |
| `height['100lvh']`      | `height: 100lvh;`                                                                     |

| `height['100svh']`      | `height: 100svh;`                                                                     |
| `height['100svw']`      | `height: 100svw;`                                                                     |
| `height['min-content']` | `height: min-content;`                                                                |
| `height['max-content']` | `height: max-content;`                                                                |
| `height['fit-content']` | `height: fit-content;`                                                                |
| `height['1lh']`         | `height: 1lh;`                                                                        |

| `size.lvh`              | `width: 100lvh;height: 100lvh;`                                                       |
| `size.svw`              | `width: 100svw;height: 100svw;`                                                       |
| `size.svh`              | `width: 100svh;height: 100svh;`                                                       |
| `size.min`              | `width: min-content;height: min-content;`                                             |
| `size.max`              | `width: max-content;height: max-content;`                                             |
| `size.fit`              | `width: fit-content;height: fit-content;`                                             |
