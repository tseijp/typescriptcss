## min-width

Utilities for setting the minimum width of an element.

| Utility                | Styles                                          |
| ---------------------- | ----------------------------------------------- |
| `min.w[*<value>*]`     | `min-width: calc(var(--spacing) * *<number>*);` |
| `min.w.auto`           | `min-width: auto;`                              |
| `min.w['1px']`         | `min-width: 1px;`                               |
| `min.w['100%']`        | `min-width: 100%;`                              |
| `min.w['100vw']`       | `min-width: 100vw;`                             |
| `min.w['100dvw']`      | `min-width: 100dvw;`                            |
| `min.w['100dvh']`      | `min-width: 100dvh;`                            |
| `min.w['100lvw']`      | `min-width: 100lvw;`                            |
| `min.w['100lvh']`      | `min-width: 100lvh;`                            |
| `min.w['100svw']`      | `min-width: 100svw;`                            |
| `min.w['100svh']`      | `min-width: 100svh;`                            |
| `min.w['min-content']` | `min-width: min-content;`                       |
| `min.w['max-content']` | `min-width: max-content;`                       |
| `min.w['fit-content']` | `min-width: fit-content;`                       |
