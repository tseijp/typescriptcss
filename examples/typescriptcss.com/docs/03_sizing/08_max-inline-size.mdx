## max-inline-size

Utilities for setting the maximum inline size of an element.

| Utility                     | Styles                                                |
| --------------------------- | ----------------------------------------------------- |
| `max.inline[*<value>*]`     | `max-inline-size: calc(var(--spacing) * *<number>*);` |
| `max.inline.auto`           | `max-inline-size: none;`                              |
| `max.inline['1px']`         | `max-inline-size: 1px;`                               |
| `max.inline['100%']`        | `max-inline-size: 100%;`                              |
| `max.inline['100dvw']`      | `max-inline-size: 100dvw;`                            |
| `max.inline['100dvh']`      | `max-inline-size: 100dvh;`                            |
| `max.inline['100lvw']`      | `max-inline-size: 100lvw;`                            |
| `max.inline['100lvh']`      | `max-inline-size: 100lvh;`                            |
| `max.inline['100svw']`      | `max-inline-size: 100svw;`                            |
| `max.inline['100svh']`      | `max-inline-size: 100svh;`                            |
| `max.inline['100vw']`       | `max-inline-size: 100vw;`                             |
| `max.inline['min-content']` | `max-inline-size: min-content;`                       |
| `max.inline['max-content']` | `max-inline-size: max-content;`                       |
| `max.inline['fit-content']` | `max-inline-size: fit-content;`                       |
