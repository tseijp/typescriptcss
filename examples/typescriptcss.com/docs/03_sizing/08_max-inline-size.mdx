## max-inline-size

Utilities for setting the maximum inline size of an element.

| Utility                   | Styles                                                |
| ------------------------- | ----------------------------------------------------- |
| `max.inline[*<value>*]`   | `max-inline-size: calc(var(--spacing) * *<number>*);` |
| `max.size.none`           | `max-inline-size: none;`                              |
| `max.size['1px']`         | `max-inline-size: 1px;`                               |
| `max.size['100%']`        | `max-inline-size: 100%;`                              |
| `max.size['100dvw']`      | `max-inline-size: 100dvw;`                            |
| `max.size['100dvh']`      | `max-inline-size: 100dvh;`                            |
| `max.size['100lvw']`      | `max-inline-size: 100lvw;`                            |
| `max.size['100lvh']`      | `max-inline-size: 100lvh;`                            |
| `max.size['100svw']`      | `max-inline-size: 100svw;`                            |
| `max.size['100svh']`      | `max-inline-size: 100svh;`                            |
| `max.size['100vw']`       | `max-inline-size: 100vw;`                             |
| `max.size['min-content']` | `max-inline-size: min-content;`                       |
| `max.size['max-content']` | `max-inline-size: max-content;`                       |
| `max.size['fit-content']` | `max-inline-size: fit-content;`                       |
