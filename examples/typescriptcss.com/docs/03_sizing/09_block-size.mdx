## block-size

Utilities for setting the block size of an element.

| Utility               | Styles                                           |
| --------------------- | ------------------------------------------------ |
| `block[*<value>*]`    | `block-size: calc(var(--spacing) * *<number>*);` |
| `size.auto`           | `block-size: auto;`                              |
| `size['1px']`         | `block-size: 1px;`                               |
| `size['100%']`        | `block-size: 100%;`                              |
| `size['100vh']`       | `block-size: 100vh;`                             |
| `size['100dvh']`      | `block-size: 100dvh;`                            |
| `size['100dvw']`      | `block-size: 100dvw;`                            |
| `size['100lvh']`      | `block-size: 100lvh;`                            |
| `size['100lvw']`      | `block-size: 100lvw;`                            |
| `size['100svh']`      | `block-size: 100svh;`                            |
| `size['100svw']`      | `block-size: 100svw;`                            |
| `size['min-content']` | `block-size: min-content;`                       |
| `size['max-content']` | `block-size: max-content;`                       |
| `size['fit-content']` | `block-size: fit-content;`                       |
| `size['1lh']`         | `block-size: 1lh;`                               |
