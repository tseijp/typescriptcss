## block-size

Utilities for setting the block size of an element.

| Utility            | Styles                                           |
| ------------------ | ------------------------------------------------ |
| `block[*<value>*]` | `block-size: calc(var(--spacing) * *<number>*);` |
| `block.auto`       | `block-size: auto;`                              |
| `block['1px']`     | `block-size: 1px;`                               |
| `block['100%']`    | `block-size: 100%;`                              |
| `block['100vh']`   | `block-size: 100vh;`                             |
| `block['100dvh']`  | `block-size: 100dvh;`                            |
| `block['100dvw']`  | `block-size: 100dvw;`                            |
| `block['100lvh']`  | `block-size: 100lvh;`                            |
| `block['100lvw']`  | `block-size: 100lvw;`                            |
| `block['100svh']`  | `block-size: 100svh;`                            |
| `block['100svw']`  | `block-size: 100svw;`                            |
| `block.min`        | `block-size: min-content;`                       |
| `block.max`        | `block-size: max-content;`                       |
| `block.fit`        | `block-size: fit-content;`                       |
| `block['1lh']`     | `block-size: 1lh;`                               |
