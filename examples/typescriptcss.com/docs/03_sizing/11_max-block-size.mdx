## max-block-size

Utilities for setting the maximum block size of an element.

| Utility                   | Styles                                               |
| ------------------------- | ---------------------------------------------------- |
| `max.block[*<value>*]`    | `max-block-size: calc(var(--spacing) * *<number>*);` |

| `max.size['1px']`         | `max-block-size: 1px;`                               |
| `max.size['100%']`        | `max-block-size: 100%;`                              |
| `max.size['100vh']`       | `max-block-size: 100vh;`                             |
| `max.size['100dvh']`      | `max-block-size: 100dvh;`                            |
| `max.size['100dvw']`      | `max-block-size: 100dvw;`                            |
| `max.size['100lvh']`      | `max-block-size: 100lvh;`                            |
| `max.size['100lvw']`      | `max-block-size: 100lvw;`                            |
| `max.size['100svh']`      | `max-block-size: 100svh;`                            |
