## max-block-size

Utilities for setting the maximum block size of an element.

| Utility                    | Styles                                               |
| -------------------------- | ---------------------------------------------------- |
| `max.block[*<value>*]`     | `max-block-size: calc(var(--spacing) * *<number>*);` |
| `max.block.none`           | `max-block-size: none;`                              |
| `max.block['1px']`         | `max-block-size: 1px;`                               |
| `max.block['100%']`        | `max-block-size: 100%;`                              |
| `max.block['100vh']`       | `max-block-size: 100vh;`                             |
| `max.block['100dvh']`      | `max-block-size: 100dvh;`                            |
| `max.block['100dvw']`      | `max-block-size: 100dvw;`                            |
| `max.block['100lvh']`      | `max-block-size: 100lvh;`                            |
| `max.block['100lvw']`      | `max-block-size: 100lvw;`                            |
| `max.block['100svh']`      | `max-block-size: 100svh;`                            |
| `max.block['100svw']`      | `max-block-size: 100svw;`                            |
| `max.block['min-content']` | `max-block-size: min-content;`                       |
| `max.block['max-content']` | `max-block-size: max-content;`                       |
| `max.block['fit-content']` | `max-block-size: fit-content;`                       |
| `max.block['1lh']`         | `max-block-size: 1lh;`                               |
