## inline-size

Utilities for setting the inline size of an element.

| Utility                 | Styles                                            |
| ----------------------- | ------------------------------------------------- |
| `inline[*<value>*]`     | `inline-size: calc(var(--spacing) * *<number>*);` |
| `inline.auto`           | `inline-size: auto;`                              |
| `inline['1px']`         | `inline-size: 1px;`                               |
| `inline['100%']`        | `inline-size: 100%;`                              |
| `inline['100vw']`       | `inline-size: 100vw;`                             |
| `inline['100dvw']`      | `inline-size: 100dvw;`                            |
| `inline['100dvh']`      | `inline-size: 100dvh;`                            |
| `inline['100lvw']`      | `inline-size: 100lvw;`                            |
| `inline['100lvh']`      | `inline-size: 100lvh;`                            |
| `inline['100svw']`      | `inline-size: 100svw;`                            |
| `inline['100svh']`      | `inline-size: 100svh;`                            |
| `inline['min-content']` | `inline-size: min-content;`                       |
| `inline['max-content']` | `inline-size: max-content;`                       |
| `inline['fit-content']` | `inline-size: fit-content;`                       |
