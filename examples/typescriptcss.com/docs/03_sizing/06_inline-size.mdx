## inline-size

Utilities for setting the inline size of an element.

| Utility               | Styles                                            |
| --------------------- | ------------------------------------------------- |
| `inline[*<value>*]`   | `inline-size: calc(var(--spacing) * *<number>*);` |
| `size.auto`           | `inline-size: auto;`                              |
| `size['1px']`         | `inline-size: 1px;`                               |
| `size['100%']`        | `inline-size: 100%;`                              |
| `size['100vw']`       | `inline-size: 100vw;`                             |
| `size['100dvw']`      | `inline-size: 100dvw;`                            |
| `size['100dvh']`      | `inline-size: 100dvh;`                            |
| `size['100lvw']`      | `inline-size: 100lvw;`                            |
| `size['100lvh']`      | `inline-size: 100lvh;`                            |
| `size['100svw']`      | `inline-size: 100svw;`                            |
| `size['100svh']`      | `inline-size: 100svh;`                            |
| `size['min-content']` | `inline-size: min-content;`                       |
| `size['max-content']` | `inline-size: max-content;`                       |
| `size['fit-content']` | `inline-size: fit-content;`                       |
