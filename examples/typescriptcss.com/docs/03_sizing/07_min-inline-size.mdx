## min-inline-size

Utilities for setting the minimum inline size of an element.

| Utility                   | Styles                                                |
| ------------------------- | ----------------------------------------------------- |
| `min.inline[*<value>*]`   | `min-inline-size: calc(var(--spacing) * *<number>*);` |
| `min.size.auto`           | `min-inline-size: auto;`                              |
| `min.size['1px']`         | `min-inline-size: 1px;`                               |
| `min.size['100%']`        | `min-inline-size: 100%;`                              |
| `min.size['100vw']`       | `min-inline-size: 100vw;`                             |
| `min.size['100dvw']`      | `min-inline-size: 100dvw;`                            |
| `min.size['100dvh']`      | `min-inline-size: 100dvh;`                            |
| `min.size['100lvw']`      | `min-inline-size: 100lvw;`                            |
| `min.size['100lvh']`      | `min-inline-size: 100lvh;`                            |
| `min.size['100svw']`      | `min-inline-size: 100svw;`                            |
| `min.size['100svh']`      | `min-inline-size: 100svh;`                            |
| `min.size['min-content']` | `min-inline-size: min-content;`                       |
| `min.size['max-content']` | `min-inline-size: max-content;`                       |
| `min.size['fit-content']` | `min-inline-size: fit-content;`                       |
