## min-inline-size

Utilities for setting the minimum inline size of an element.

| Utility                 | Styles                                                |
| ----------------------- | ----------------------------------------------------- |
| `min.inline[*<value>*]` | `min-inline-size: calc(var(--spacing) * *<number>*);` |
| `min.inline.auto`       | `min-inline-size: auto;`                              |
| `min.inline['1px']`     | `min-inline-size: 1px;`                               |
| `min.inline['100%']`    | `min-inline-size: 100%;`                              |
| `min.inline['100vw']`   | `min-inline-size: 100vw;`                             |
| `min.inline['100dvw']`  | `min-inline-size: 100dvw;`                            |
| `min.inline['100dvh']`  | `min-inline-size: 100dvh;`                            |
| `min.inline['100lvw']`  | `min-inline-size: 100lvw;`                            |
| `min.inline['100lvh']`  | `min-inline-size: 100lvh;`                            |
| `min.inline['100svw']`  | `min-inline-size: 100svw;`                            |
| `min.inline['100svh']`  | `min-inline-size: 100svh;`                            |
| `min.inline.min`        | `min-inline-size: min-content;`                       |
| `min.inline.max`        | `min-inline-size: max-content;`                       |
| `min.inline.fit`        | `min-inline-size: fit-content;`                       |
