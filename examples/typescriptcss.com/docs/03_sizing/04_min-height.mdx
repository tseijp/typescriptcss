## min-height

Utilities for setting the minimum height of an element.

| Utility            | Styles                                           |
| ------------------ | ------------------------------------------------ |
| `min.h[*<value>*]` | `min-height: calc(var(--spacing) * *<number>*);` |
| `min.h['1px']`     | `min-height: 1px;`                               |
| `min.h['100%']`    | `min-height: 100%;`                              |
| `min.h['100vh']`   | `min-height: 100vh;`                             |
| `min.h['100dvh']`  | `min-height: 100dvh;`                            |
| `min.h['100dvw']`  | `min-height: 100dvw;`                            |
| `min.h['100lvh']`  | `min-height: 100lvh;`                            |
| `min.h['100lvw']`  | `min-height: 100lvw;`                            |
| `min.h['100svw']`  | `min-height: 100svw;`                            |
| `min.h['100svh']`  | `min-height: 100svh;`                            |
| `min.h.auto`       | `min-height: auto;`                              |
| `min.h.min`        | `min-height: min-content;`                       |
| `min.h.max`        | `min-height: max-content;`                       |
| `min.h.fit`        | `min-height: fit-content;`                       |
| `min.h['1lh']`     | `min-height: 1lh;`                               |
