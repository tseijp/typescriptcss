## padding

Utilities for controlling an element's padding.

| Utility          | Styles                                                     |
| ---------------- | ---------------------------------------------------------- |
| `p[*<value>*]`   | `padding: calc(var(--spacing) * *<number>*);`              |
| `padding['1px']` | `padding: 1px;`                                            |
| `px[*<value>*]`  | `padding-inline: calc(var(--spacing) * *<number>*);`       |
| `px['1px']`      | `padding-inline: 1px;`                                     |
| `py[*<value>*]`  | `padding-block: calc(var(--spacing) * *<number>*);`        |
| `py['1px']`      | `padding-block: 1px;`                                      |
| `ps[*<value>*]`  | `padding-inline-start: calc(var(--spacing) * *<number>*);` |
| `ps['1px']`      | `padding-inline-start: 1px;`                               |
| `pe[*<value>*]`  | `padding-inline-end: calc(var(--spacing) * *<number>*);`   |
| `pe['1px']`      | `padding-inline-end: 1px;`                                 |
| `pbs[*<value>*]` | `padding-block-start: calc(var(--spacing) * *<number>*);`  |
| `pt['1px']`      | `padding-block-start: 1px;`                                |
| `pbe[*<value>*]` | `padding-block-end: calc(var(--spacing) * *<number>*);`    |
| `pb['1px']`      | `padding-block-end: 1px;`                                  |
| `pt[*<value>*]`  | `padding-top: calc(var(--spacing) * *<number>*);`          |
| `pr[*<value>*]`  | `padding-right: calc(var(--spacing) * *<number>*);`        |
| `pr['1px']`      | `padding-right: 1px;`                                      |
| `pb[*<value>*]`  | `padding-bottom: calc(var(--spacing) * *<number>*);`       |
| `pl[*<value>*]`  | `padding-left: calc(var(--spacing) * *<number>*);`         |
| `pl['1px']`      | `padding-left: 1px;`                                       |
