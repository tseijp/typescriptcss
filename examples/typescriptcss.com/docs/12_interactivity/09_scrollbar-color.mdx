## scrollbar-color

Utilities for controlling the color of an element's scrollbar.

| Utility                       | Styles                                 |
| ----------------------------- | -------------------------------------- |
| `scrollbar.thumb.inherit`     | `scrollbar-color: inherit;`            |
| `scrollbar.thumb.current`     | `scrollbar-color: currentColor;`       |
| `scrollbar.thumb.transparent` | `scrollbar-color: transparent;`        |
| `scrollbar.thumb.black`       | `scrollbar-color: var(--color-black);` |
| `scrollbar.thumb.white`       | `scrollbar-color: var(--color-white);` |
| `scrollbar.thumb[*<value>*]`  | `scrollbar-color: *<value>*;`          |
| `scrollbar.track[*<value>*]`  | `scrollbar-color: *<value>*;`          |
