## scrollbar-width

Utilities for controlling the width of an element's scrollbar.

| Utility          | Styles                   |
| ---------------- | ------------------------ |
| `scrollbar.auto` | `scrollbar-width: auto;` |
| `scrollbar.thin` | `scrollbar-width: thin;` |
| `scrollbar.none` | `scrollbar-width: none;` |
