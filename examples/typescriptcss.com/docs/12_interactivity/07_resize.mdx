## resize

Utilities for controlling how an element can be resized.

| Utility       | Styles                |
| ------------- | --------------------- |
| `resize.none` | `resize: none;`       |
| `resize`      | `resize: both;`       |
| `resize.y`    | `resize: vertical;`   |
| `resize.x`    | `resize: horizontal;` |
