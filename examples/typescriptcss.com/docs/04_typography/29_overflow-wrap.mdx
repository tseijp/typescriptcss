## overflow-wrap

Utilities for controlling line breaks within words in an overflowing element.

| Utility              | Styles                       |
| -------------------- | ---------------------------- |
| `wrap['break-word']` | `overflow-wrap: break-word;` |
| `wrap.anywhere`      | `overflow-wrap: anywhere;`   |
| `wrap.normal`        | `overflow-wrap: normal;`     |
