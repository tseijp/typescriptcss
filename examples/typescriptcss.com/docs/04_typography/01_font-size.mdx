## font-size

Utilities for controlling the font size of an element.

| Utility           | Styles                                                                                                           |
| ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| `text.base`       | `font-size: var(--text-base); /* 1rem (16px) */ line-height: var(--text-base--line-height); /* calc(1.5 / 1) */` |
| `text[*<value>*]` | `font-size: var(*<custom-property>*);`                                                                           |
