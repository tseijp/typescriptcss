## text-overflow

Utilities for controlling how the text of an element overflows.

| Utility         | Styles                                                          |
| --------------- | --------------------------------------------------------------- |
| `truncate`      | `overflow: hidden;text-overflow: ellipsis;white-space: nowrap;` |
| `text.ellipsis` | `text-overflow: ellipsis;`                                      |
| `text.clip`     | `text-overflow: clip;`                                          |
