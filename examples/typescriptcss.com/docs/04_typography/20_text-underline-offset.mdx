## text-underline-offset

Utilities for controlling the offset of a text underline.

| Utility                       | Styles                                 |
| ----------------------------- | -------------------------------------- |
| `underline.offset[*<value>*]` | `text-underline-offset: *<number>*px;` |
| `underline.offset.auto`       | `text-underline-offset: auto;`         |
