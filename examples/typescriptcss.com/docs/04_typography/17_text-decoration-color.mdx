## text-decoration-color

Utilities for controlling the color of text decorations.

| Utility                   | Styles                                                  |
| ------------------------- | ------------------------------------------------------- |
| `decoration.inherit`      | `text-decoration-color: inherit;`                       |
| `decoration.currentColor` | `text-decoration-color: currentColor;`                  |
| `decoration.transparent`  | `text-decoration-color: transparent;`                   |
| `decoration.black`        | `text-decoration-color: var(--color-black); /* #000 */` |
| `decoration.white`        | `text-decoration-color: var(--color-white); /* #fff */` |
| `decoration[*<value>*]`   | `text-decoration-color: var(*<custom-property>*);`      |
