## word-break

Utilities for controlling word breaks in an element.

| Utility        | Styles                   |
| -------------- | ------------------------ |
| `break.normal` | `word-break: normal;`    |
| `break.all`    | `word-break: break-all;` |
| `break.keep`   | `word-break: keep-all;`  |
