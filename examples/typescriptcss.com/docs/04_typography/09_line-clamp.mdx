## line-clamp

Utilities for clamping text to a specific number of lines.

| Utility                 | Styles                                                                                               |
| ----------------------- | ---------------------------------------------------------------------------------------------------- |
| `line.clamp[*<value>*]` | `overflow: hidden;display: -webkit-box;-webkit-box-orient: vertical;-webkit-line-clamp: *<number>*;` |
| `line.clamp.none`       | `overflow: visible;display: block;-webkit-box-orient: horizontal;-webkit-line-clamp: unset;`         |
