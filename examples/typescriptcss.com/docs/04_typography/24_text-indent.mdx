## text-indent

Utilities for controlling the amount of empty space shown before text in a block.

| Utility             | Styles                                            |
| ------------------- | ------------------------------------------------- |
| `indent[*<value>*]` | `text-indent: calc(var(--spacing) * *<number>*);` |
| `indent['1px']`     | `text-indent: 1px;`                               |
| `indent['-1px']`    | `text-indent: -1px;`                              |
