## text-wrap

Utilities for controlling how text wraps within an element.

| Utility        | Styles                |
| -------------- | --------------------- |
| `text.wrap`    | `text-wrap: wrap;`    |
| `text.nowrap`  | `text-wrap: nowrap;`  |
| `text.balance` | `text-wrap: balance;` |
| `text.pretty`  | `text-wrap: pretty;`  |
