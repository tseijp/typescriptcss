## list-style-image

Utilities for controlling the marker images for list items.

| Utility                 | Styles                         |
| ----------------------- | ------------------------------ |
| `list.image[*<value>*]` | `list-style-image: *<value>*;` |
| `list.image.none`       | `list-style-image: none;`      |
