## list-style-type

Utilities for controlling the marker style of a list.

| Utility           | Styles                                       |
| ----------------- | -------------------------------------------- |
| `list.disc`       | `list-style-type: disc;`                     |
| `list.decimal`    | `list-style-type: decimal;`                  |
| `list.none`       | `list-style-type: none;`                     |
| `list[*<value>*]` | `list-style-type: var(*<custom-property>*);` |
