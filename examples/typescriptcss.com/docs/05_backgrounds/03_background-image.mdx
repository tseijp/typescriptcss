## background-image

Utilities for controlling an element's background image.

| Utility                 | Styles                                                                                    |
| ----------------------- | ----------------------------------------------------------------------------------------- |
| `bg[*<value>*]`         | `background-image: *<value>*;`                                                            |
| `bg.none`               | `background-image: none;`                                                                 |
